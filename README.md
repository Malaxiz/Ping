# Ping

### Usage


| Code           |   Explanation                                                |
| -------------  |:-------------:                                               |
| `import ping`  | Imports the module                                           |
| `generate_ips` | Generate a range of local ips                                |
| `start`        | Starts pinging all the ip addresses                          |
| `print_nice`   | Prints the return value of `start` in a human readable way   |
